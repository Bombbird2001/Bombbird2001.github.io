Clear Storage

If you have received an error message during loading of the game, do the following steps:

1. Click any region outside of the game canvas

2. Press F12 on your keyboard

3. Select the "Console" tab

4. Type the following code into the console (at where the blue > or >> is located): localStorage.clear();

5. Press Enter (You should see the console return "undefined")

6. Storage is now cleared, all game data has been reset

7. Refresh the page
